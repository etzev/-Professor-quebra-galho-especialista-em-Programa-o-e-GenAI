# Fundamentos da Programação

Os materiais detalham conceitos fundamentais como:

- Variáveis
- Tipos de dados
- Estruturas de repetição
- Estruturas condicionais

Esses conceitos são exemplificados por meio de linguagens amplamente utilizadas no mercado, como:
- Python
- Java
- C#

A abordagem prática permite compreender a lógica computacional
independente da linguagem utilizada.
